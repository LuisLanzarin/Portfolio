Projeto 1: Portfolio

Trabalhando com HTML, CSS e JavaScript

Este é meu portfólio pessoal desenvolvido para apresentar minhas habilidades, experiências e projetos. O site é dividido em seções como Home, Sobre Mim, Currículo e Portfólio. Cada projeto conta com imagens e descrições curtas, destacando meu progresso na programação. O layout é limpo, moderno e responsivo, garantindo boa navegação em qualquer dispositivo. O portfólio foi desenvolvido com HTML, CSS e JavaScript puro. Está hospedado na Vercel com foco em performance e usabilidade.
