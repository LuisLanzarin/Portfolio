Projeto 1: Portfolio

Trabalhando com HTML, CSS e JavaScript

Este é meu portfólio pessoal desenvolvido para apresentar minhas habilidades, experiências e projetos. O site é dividido em seções como Home, Sobre Mim, Currículo e Portfólio. Projeto 2: Alura+

Replicando possível site Alura+

Este projeto é uma landing page responsiva criada com HTML e CSS, inspirada no serviço de streaming Alura+. O site simula uma página promocional com foco na assinatura dos planos. Possui seções bem definidas, botões de ação e design profissional. Foi desenvolvido com atenção à responsividade, funcionando bem em dispositivos móveis e desktops. O layout destaca planos combinados entre Alura+ e Alura Língua, com textos e imagens bem alinhados. Ideal para treinar estruturação de páginas modernas e uso de CSS Flexbox.
